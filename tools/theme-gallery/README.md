# Native SDK Theme Gallery

A tiny Svelte gallery for the theme modules in `../../themes`.

```sh
npm install
npm run dev
```

Install every theme into an app-local `themes/` directory:

```sh
curl -fsSL https://raw.githubusercontent.com/henryoman/awesome-native-sdk/main/scripts/install-theme.sh | sh -s -- all
```

This is intentionally separate from the repo root. It is a basic preview/index, not the source of truth for theme tokens.
